# 107 Studio Website

Production website for 107 Studio built with Next.js.

## Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm run start
```

## Static Share Build

This project uses static export output.
After build, share or upload the `out/` directory (for example in Netlify Drop).
